# 模拟数据验收基线 v1

## 验收目标
- 保证模拟数据在“范围、趋势、相关性、异常比例”上可解释
- 保证同一批数据可同时服务监控预警、预处理、训练与可视化

## 必验指标

| 类别 | 指标 | 建议阈值 |
|---|---|---|
| 完整性 | 缺失率（除时间戳） | ≤ 2% |
| 完整性 | 哨兵值占比（9999*） | ≤ 1% |
| 范围性 | 硬限制越界率 | ≤ 0.5% |
| 范围性 | 预警带触发率 | 2% ~ 10% |
| 稳定性 | 单参数异常尖峰连续点数 | 不超过 3 点 |
| 相关性 | windVolume 与 productionRate | 正相关 |
| 相关性 | energyConsumption 与 siliconContent | 正相关 |
| 时序性 | 相邻点波动率 | 不出现长时间常量段 |

## 推荐采样包
- 样本数：至少 2000 条
- 采样周期：10s 或 30s
- 覆盖工况：装料、送风、冶炼、出铁、异常

## 答辩表述建议
- 先定义统一契约，再校准模拟器，最后验证多模块可用
- 强调“可解释工业合理性”而非“绝对真实工业机理”
